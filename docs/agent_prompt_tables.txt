⚠️  Before Finalising Any File
1. Does every esp_err_t return get checked? (No silent failures)
2. Are all FreeRTOS tasks registered with the task watchdog?
3. Are secrets absent from source — only read from NVS?
4. Does the Zigbee cluster attribute list match the data model structs?
5. Do CMakeLists REQUIRES lists include ALL used IDF components?
6. Is deep-sleep entry guarded by a pending-Zigbee-TX check?
7. Does OTA verify SHA-256 + RSA signature before boot switch?

=== TABLE 15 ===
🚫  Hard Rules — Never Violate
• NO placeholder functions: void my_func() { /* TODO */ } is forbidden
• NO Arduino framework: pure ESP-IDF idf.py only
• NO blocking delays in ISR context — use xQueueSendFromISR()
• NO floating-point in ISR — convert to integer fixed-point before ISR
• NO credentials or keys hard-coded in source — NVS only
• NO unchecked malloc() — always verify != NULL before use
• NO task stacks below 4096 bytes — increase if unsure
• NO printf() — use ESP_LOGx macros only
• ALL Zigbee callbacks must return within 5 ms — offload to queues
• ALL sensor drivers must handle I2C NACK / UART timeout gracefully
