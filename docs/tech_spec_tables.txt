=== TABLE 13 ===
🔒  eFuse Configuration (Production)
• JTAG disable bit: burn to prevent debug access on production units
• Secure download mode: restrict UART flashing after production provisioning
• Key block 0: Secure Boot RSA-2048 public key digest
• Key block 1: Flash Encryption AES-256 key (hardware generated)
• WR_DIS bits: write-protect all sensitive eFuse blocks after provisioning
